# 포트폴리오 스토리

## 이 프로젝트에서 보여주고 싶은 것
MockStock Live는 단순한 모의주식 서비스가 아닙니다.
이 프로젝트를 통해 아래를 보여주고자 함.

- 거래 데이터 정합성
- 실시간 네트워크 설계 능력
- 조회 성능 개선 능력
- 병목을 측정하고 개선하는 방식
- 운영 지표 관측 능력

## 가장 중요한 문제 해결 사례
1. 채팅방 목록 조회 병목
2. 거래 내역 pagination 성능 저하
3. 메시지 조회 offset 문제
4. 매수/매도 동시성 정합성 문제
5. 실시간 연결 상태 가시성 부족 문제

## 결과적으로 얻은 것
- 기능 구현을 넘어서, 어떤 구조가 왜 느려지는지 설명할 수 있게 되었다.
- 단순히 “최적화했다”가 아니라, 실행 계획과 지표 기반으로 개선을 증명할 수 있게 되었다.
